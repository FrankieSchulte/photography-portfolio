# Design notes

The current direction is a Y2K fashion-editorial image archive rather than a conventional boxed portfolio template.

- The landing page is one viewport and does not become a long marketing scroll.
- Several highlighted photographs appear immediately in a horizontally scrollable, snap-aligned sequence.
- Oversized condensed typography, fine grid lines, acid/cobalt accents, ASCII marks, barcodes, and registration details create the magazine system.
- Frames and controls use sharp print geometry; photography remains full color and visually dominant.
- Work categories are equal in hierarchy; the site does not present Frankie as exclusively a graduation, concert, or event photographer.
- Categories contain shoots rather than a single undifferentiated thumbnail wall.
- Shoot pages use an asymmetric editorial sequence and an accessible lightbox.
- Work-category and shoot-cover boxes are square. Full shoot galleries remain 3:2 or 2:3.
- Motion is subtle, stops when the page is hidden, and is removed when `prefers-reduced-motion` is enabled.
